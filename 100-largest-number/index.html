Add the JS code to the script.js file.

Step 1

  1. Write a function that takes the array as argument
  2. The function must return the largest number within the array
  3. Use a default for-loop to solve that exercise
  4. Watch out for edge cases: empty arrays or null values
  5. If no maximum was found return undefined
  6. print the found maximums to the console with console.log()

Step 2

  1. Add another function that also finds the biggest number named getMaximumByReducing()
  2. Instead of using a for loop try to use the array function reduce()
  3. Try to find out how reduce works by your own!
  4. Print the found maximums to the console with console.log() for that implementation as well
