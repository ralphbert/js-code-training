Add the JS code to the script.js file.

  1. When the user clicks the button the system picks a random number between 0 and 100
  2. Now the user must guess the number via prompt()-inputs
  3. If the number is too big or too small the user will get a corresponding hint, like "Number too big" or "Number too small" via alert()
  4. The game ends when the user guesses the right number
  5. The game should present the statistics of the game round in the div#output element
  6. It must contain all guesses and the number of total guesses needed

Example

System picks the number 5

User input via prompt = 10
Alert shown: Number too big.

User input via prompt = 2
Alert shown: Number too small.

User input via prompt = 6
Alert shown: Number too big.

User input via prompt = 5
Alert shown: WINNER! The Number was 5

Output:
1) 10
2) 2
3) 6
4) 5
Total guesses needed: 4
