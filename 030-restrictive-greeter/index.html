Restrictive greeter

Add the JS code to the script.js file.

  1. When you click the button the browser should ask you for your name. Use the prompt() function to do that.
  2. Only the names "Bob" or "Marie" are allowed! Watch out for case sensitivity: "bob", "MARie", "bOB" should all be valid inputs!
  3. The output should be like: Welcome NAME!. Example: The user enters "Marie" the output should be "Welcome Marie!"
  4. Make sure that the names are transformed to a capital starting letter followed by lower case letters: "BOB" --> "Bob", "marie" --> "Marie". Find a way to do that. Create a function that takes the name (like "BOB") and returns the modified wellformed name ("Bob") as a return value.
  5. If a forbidden name is entered the output should be "Forbidden name: NAME!"
