Number printer

Add the JS code to the script.js file.

  1. When the user clicks the button ask for a positive number (named "n")
  2. If the user inputs a negative or any other invalid number (like text or a decimal) show an alert that the input was wrong
  3. If the number is greater than 100 set it to a maximum of 100. Use Math.min() to do that
  4. Print all numbers to the div with the id named output from 1 to n.

Example

Input = 5
1
2
3
4
5
